# UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

Description: 
This project is a MySQL-based E-Commerce Database Management System designed to manage the core operations of an online shopping platform. It includes modules for customer details, product and stock management, order processing, payments, and shipping tracking. The system uses relational database concepts with primary and foreign keys to maintain data integrity. Various SQL operations such as CREATE, INSERT, SELECT, UPDATE, DELETE, and ALTER are implemented. The project demonstrates real-world database design, normalization, and CRUD operations using MySQL. It is ideal for understanding how e-commerce platforms manage structured data efficiently. Give banking
